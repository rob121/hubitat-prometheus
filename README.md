# Hubitat Prometheus

A prometheus collector for hubitat

## Configuration

```
{
  "access_token": "d7bcc9daa2f8e",
  "hubitat_base_url": "http://192.168.20.5/apps/api/12",
  "hubitat_ip": "192.168.20.5"
}
```
